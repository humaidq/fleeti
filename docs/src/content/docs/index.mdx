## Why Fleeti

<CardGrid stagger>
	<Card title="Declarative configuration" icon="setting">
		Define packages, services, users, and security policies for your fleet through an easy-to-use web interface.
	</Card>
	<Card title="Reproducible builds" icon="puzzle">
		Nix-powered build pipeline produces deterministic, versioned images every time. Same config, same output.
	</Card>
	<Card title="Secure by design" icon="approve-check-circle">

	</Card>
	<Card title="Fleet-scale rollouts" icon="rocket">

	</Card>
