# Changelog

All notable Fleeti releases are tracked here.

## [v0.2.0] - 2026-03-27

- Add HTTP APIs for profiles, builds, and API-key auth alongside broader permission controls for fleets and profiles.
- Introduce profile wizard, security search, and kernel patch upload flows to improve profile authoring and hardening.
- Improve deployment and dashboard UX, including public-facing pages, live build logs, and deployment views.
- Embed the NixOS base image and reorganize Nix image and update plumbing under `src/nixos/`.
- Add project docs, the docs overlay, and release-facing version metadata in the CLI build.
